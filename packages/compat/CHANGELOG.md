# @dzip-ui/compat

## 0.1.0-alpha.0 (2026-04-06)

### Features

- Initial alpha release of dzip-ui migration compatibility adapters
- Backward-compatible wrapper components for migrating from legacy `dzip-ui` API
- Compat adapters: DzButtonCompat, DzInputCompat, DzSelectCompat, DzCheckboxCompat, DzRadioCompat, DzSwitchCompat, DzDialogCompat, DzAccordionCompat, DzTabsCompat, DzTooltipCompat, DzAlertCompat
- Deprecation warning utilities (`warnDeprecated`, `resetDeprecationWarnings`)
