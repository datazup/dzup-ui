# @dzip-ui/nuxt

## 0.1.0-alpha.0 (2026-04-06)

### Features

- Initial alpha release of the dzip-ui Nuxt module
- Auto-imports all `Dz*` components from `@dzip-ui/core`
- Optional `includePro` flag to auto-import `@dzip-ui/pro` components
- Configurable component prefix via `prefix` option
- Compatible with Nuxt 3.x and `@nuxt/kit`
