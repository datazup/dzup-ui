# dzup-ui Component Library

Welcome to the **dzup-ui** component library documentation — 147 foundational Vue 3 components built with TypeScript, Tailwind CSS 4, and design tokens.

> **Enterprise components** (Kanban, Gantt, FormBuilder, WorkflowDesigner, etc.) are available in [@dzup-ui/pro](https://github.com/datazup/dzup-ui-pro) under a commercial license.

## Architecture

- **Vue 3.5+** with Composition API and `<script setup lang="ts">`
- **TypeScript** with strict mode and full type coverage
- **Tailwind CSS 4** with CSS-first design tokens
- **tailwind-variants (tv)** for type-safe variant styling
- **Reka UI** for headless interaction primitives
- **OKLCH color system** for perceptually uniform colors

## Design Tokens

All components use a three-tier token system:

1. **Primitive** — raw palette and scale values (OKLCH colors)
2. **Semantic** — intent-based mappings that adapt to light/dark themes
3. **Component** — per-component overrides

Use the **Theme** toolbar toggle above to switch between light and dark modes.

## Contract Spec

Every component follows the Contract Spec v1 with canonical prop groups:

- **Appearance**: `size` (xs/sm/md/lg/xl), `variant`, `tone`
- **Behavior**: `disabled`, `loading`, `readonly`
- **State**: `invalid`, `error`, `required`
- **Accessibility**: `id`, `ariaLabel`, `ariaLabelledby`
